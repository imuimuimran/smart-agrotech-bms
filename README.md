# Smart AgroTech Business Management System (BMS)

## Overview

The Smart AgroTech Business Management System (BMS) is a full-stack web application designed to streamline agricultural machinery business operations.

## Technology Stack

### Frontend

- React (Vite)
- Tailwind CSS
- React Router
- TanStack Query

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Core Modules

- Authentication
- Dashboard
- User Management
- Product Management
- Category Management
- Supplier Management
- Purchase Management
- Inventory Management
- Customer Management
- Sales Management
- Customer Due Management
- Supplier Due Management
- Expense Management
- Reporting
- Activity Logs
- Notifications
